## 🔧 Usage Instructions

Once you have installed clrun, it’s time to get started with some basic commands:

- **Start an AI Session:** 
   ```
   clrun start [model_name]
   ```
   Replace `[model_name]` with the AI model you would like to utilize (e.g., `claude`, `moltbot`).

- **List Available Models:**
   ```
   clrun list
   ```

- **Get Help:**
   ```
   clrun help
   ```
   This command provides a summary of available commands and options.
